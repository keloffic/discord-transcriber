
# Discord Transcriber Bot

Este bot se conecta a tu servidor de Discord y puede transcribir audios usando Whisper. El flujo est√° pensado para enviar los resultados a n8n.

## Requisitos

- Node.js 18 o superior
- Token de Bot de Discord
- API Key de OpenAI
- Webhook de n8n (opcional)

## Uso

1. Clona este repositorio.
2. Renombra `.env.example` a `.env` y coloca tus claves.
3. Instala dependencias: `npm install`
4. Inicia el bot: `npm start`
